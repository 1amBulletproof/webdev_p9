BHC would like it's employees to be able to view this database from a web page. They would like you to create a web page that lets the user view all entries in the database that are equal to or after a given start date.

So, you will need to create a form that will accept a year, month and day, and then query the database. The data returned should be formatted into an html table and shown to the user. It should contain the dates, the location of the tour, the guides name, and the first and last name on the reservation. Note this is based on a start day only, there is no end day. Do not assume that input/output is Java based, use simple text streams (No Java Objects).

Oh, and of course you'll need to connect to the database. You can manually connect to the database if you are logged in to the class computer with the parameters as in the command:

/usr/bin/mysql -u johncolter -pLetMeIn class

The gives you read-only access to the database.

To actually develop your application, you'll need to make a new database connection.

  1. Go to the top left window of Netbeans and click on Services
  2. Right click on Databases and select "New Connection".Note that you may need to select the driver you want to use. Look for the MySQL driver.
  3. Fill in the next panel as shown below and select the Test Connection button to make sure connection succeeds. If so you will see the following display.

This will now give you a connection to the MySQL database. You can make a connection with the URL

String dbURL = "jdbc:<database-name>://" + host + ":" + port + "/" + dbName;

Where the database name is mysql, the host is web6.jhuep.com (or whatever the course computer is) and the port is 3306 and the dbName is "class".
